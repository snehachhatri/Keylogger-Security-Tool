# 🔐 Keylogger Security Tool

A Python-based educational security tool that demonstrates how keyloggers work — records keypresses, saves to file with timestamps, and sends automatic email reports.

---

## ⚠️ Disclaimer
This tool is for **educational purposes only**.
Use only on your own system. Unauthorized use is illegal.

---

## ⚡ Features
- Records all keypresses with timestamp
- Captures special keys (Enter, Backspace, Caps Lock etc.)
- Saves logs to local file automatically
- Sends email report every 60 seconds

---

## 🛠️ Tech Stack
- Python 3
- pynput — keyboard monitoring
- smtplib — email functionality
- threading — background email timer
- datetime — timestamps

---

## ⚙️ Requirements
```bash
pip install pynput
```

---

## 🚀 How to Run
```bash
python keylogger.py
```
Press **ESC** to stop.
